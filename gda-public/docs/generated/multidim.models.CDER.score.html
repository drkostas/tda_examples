multidim.models.CDER.score¶

CDER.score(pointclouds, labels)¶

Score predicted labels against known labels.

Parameters:
pointclouds : list-like

A list of arrays. Each array is a pointcloud. Each row of that array is a point.

labels : list-like

A corresponding list of labels.
Returns:
an array of booleans. True means the labels match.
