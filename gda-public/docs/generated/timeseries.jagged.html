timeseries.jagged¶

timeseries.jagged(persdiag, index)[source]¶

Produce a piecewise-linear function that matches the given persistence diagram. This assumes that the index for the vertices is sequential and linear, so that the mergetree can be ignored.

Parameters:
persdiag : homology.PersDiag

A 0-dimensional persistence diagram

index : list-like

The domain index for the function
Returns:
function : pandas.Series
