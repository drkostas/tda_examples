multidim.covertree¶

This module contains the essential classes for the “Cover-tree with friends” algorithm, namely:

  • CoverTree
  • CoverLevel

This module also defines the constants

  • ratio_Ag \(=\sqrt{2} - 1=0.414\ldots\), the inverse of the silver ratio
  • ratio_Au \(=\frac{\sqrt{5} - 1}{2}=0.618\ldots\), the inverse of the golden ratio
